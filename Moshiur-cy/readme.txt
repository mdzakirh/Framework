
Prerequisite: Must need node.js latest version installed in your machine. (v13.x or grater)

Step 1: Install all the dependencies. How: run command 'npm install' in the root directory. 

Step 2: Fill the configuration in the ./cypress/config/base-config.js file. 

Step 3: run command 'npx cypress open' or 'npx cypress run' for test execution.
